Spring 2017 Responsive Design Final

Independently redesign the China Star Dim Sum & Lounge restaurant website.

Project Background

One of my favorite local restaurants is China Star Dim Sum & Lounge. The food is authentic Chinese, there's free parking, and it's only one of a handful of restaurants to offer Dim Sum outside of Boston. As delectable as this place's food is, their website leaves a lot to be desired.

This restaurant unfortunately shares its cuisine and name with two other establishments (one in the same city no less!). The current site does little to distinguish it from the pack, so I chose to take on this challenge as my Responsive Design class' final project.

My Role

Research, conceptualize, design, and develop the website with no interaction and direction from the client.

Project Goals

  • Responsive layout, mobile first
  • Distinguish branding
  • Emphasize the Dim Sum
  • Easily accessible navigation and contact info
  • Personal Goal: leverage flexbox

Technologies

  • HTML5
  • CSS3
  • CSS3 Flexbox
  • CSS-only mobile navigation show/hide
  • GitHub
  • Sass
  • Excel for transcribing menu items

Style Tile Variations

These Style Tile all focused on emphasizing images of the Dim Sum because it would help distinguish the restaurant. All the color palettes use the traditional Chinese color scheme of reds and golds, but each explores the effect of varying the shades, primary colors, and fonts.

Final Style Tile

The final Style Tile combined elements from the first and third variations. From Tile 1: a strong hero image with a transparent caption overlay would appear at the top of every page. And from Tile 3: the darker color palette would emphasize the photos without competing with them.

Excel Formulas for Organization

The takeout menu I referenced contained 436 dishes, all of which needed to be organized and visible on the new site. First, I used Excel to manage the menu's multiple categories and massive listings.

Excel Pivot Tables to Transcribe

When it came time to transfer the data from Excel into the HTML files, I used Pivot Tables on a separate sheet to quickly filter categories and show only the final formulated HTML code for a simple copy/paste.
