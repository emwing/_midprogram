Fall 2017 Web Development Week 7 Exercise

An exercise demonstrating a JavaScript onClick function that applies a CSS3 style with transitions.

Exercise 7.2: Trading Classes

Midway through the current semester, the Web Development course entered week 7, and the fifth week of learning JavaScript. The week's topic focused on three key functions of JavaScript that I had been dying to learn since starting the course:

  1. Accessing the DOM with JavaScript
  2. Modifying the DOM with JavaScript
  3. Changing the appearance of DOM elements by managing CSS classes with JavaScript

My Role

Design and develop a basic website with an onClick JavaScript handler.

Exercise Requirements

  • Create an HTML document that contains a DIV with an ID attribute
  • In an external CSS stylesheet, apply styles to this DIV element. Also create a CSS class that would change the appearance of the DIV when the class is applied to the DIV
  • In an external JavaScript file, use an onClick handler to apply the CSS class to the DIV
  • Then, try experimenting with CSS3 transitions on the property that is changed by adding the class

Technologies

  • HTML5
  • CSS3
  • JavaScript onClick handler
  • GitHub
